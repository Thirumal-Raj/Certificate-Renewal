# Certificate Renewal

A project for managing and automating certificate renewal processes.

## Structure

```
Certificate_Renewal/
├── README.md
├── src/
│   └── main.py
├── config/
│   └── config.yaml
├── requirements.txt
└── .gitignore
```

## Getting Started

1. Install dependencies: `pip install -r requirements.txt`
2. Configure settings in `config/config.yaml`
3. Run: `python src/main.py`
